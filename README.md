## Описание

Приложение представляет собой интернет-магазин аренды строительных инструментов.
Пользователь может просматривать каталог доступного инструмента, открывать
карточки товаров с описанием и оформлять заказ на аренду.

## Возможности

- Каталог инструментов с разбивкой по категориям
- Карточка товара с описанием, ценой и условиями аренды
- Оформление заказа на аренду
- Хранение данных в облаке Firebase
- Отправка заявок на email через EmailJS

> Отредактируйте список под реальный функционал вашего приложения.

## Технологии

- **Язык:** Kotlin
- **Платформа:** Android (Android Studio)
- **Backend и база данных:** Firebase
- **Отправка email:** EmailJS
- **Интерфейс:** Android SDK

## Как запустить

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/Yojin12387/Rental-of-tools.git
   ```
2. Открыть проект в Android Studio.
3. Дождаться синхронизации Gradle.
4. Запустить на эмуляторе или физическом устройстве (кнопка **Run ▶**).
